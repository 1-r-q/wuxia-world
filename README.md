# 이 무림에 낙원은 없다 (此武林無桃源)

무협 세계관 소개 웹사이트 - 강호의 영웅과 세력들을 만나보세요.

## 기술 스택

- **Framework**: Next.js 16
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4
- **Animation**: Framer Motion
- **Audio**: Howler.js
- **Smooth Scroll**: Lenis

## 시작하기

`ash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 프로덕션 서버 실행
npm start
`
